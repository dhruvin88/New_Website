Master's Project: Transfer Learning in CNNs with Medical Images

For my Master's project, I'm testing if pretained image models trained on ImageNet can help improve performance in specific medical image datasets. I will be testing which layers and how many layers in a CNN are generalize or specific to a task. Also, testing to see if using a pretained model with small amount of training data can outperfrom a model that is only trained on that task.
